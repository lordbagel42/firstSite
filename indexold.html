Docker

Docker is a software used for containerization, a process where you keep apps in little sandboxes. This means they can't mess with each other, along with very simple installation.

Coder

Coder is a software for making "ephemeral workspaces", basically, it allows for me to develop inside the cloud instead of on a local machine. This makes anything I do always up to date, and easily recreatable and shareable.

I actually switched to developing this in coder soon after I finished this paragraph. It works a lot better.
Nginx Proxy Manager

Nginx is a software for reverse proxying. This means I can have multiple pieces of software go over the same port, and lead to different subdomains (the * part of *.example.com). It also allows security.
Portainer

Portainer is a graphical interface for Docker. I wish I had not used it. It has many useful features, however, it also tends to cause some issues.

The largest issue with Portainer that I face is I cannot change any of the configuration files on my server unless I have my computer running, as that's where the Portainer dashboard is.
